# Shadow Bounce - Multi World

A sleek, browser-based paddle ball game with multiple visually stunning worlds and addictive gameplay. Keep the ball bouncing on your paddle for as long as possible and aim for the highest score!

---

## Features

- **Multiple Worlds:**  
  - **World 1 - Classic Shadow:** A cool blue glowing shadow theme.  
  - **World 2 - Gold World:** Warm gold hues with subtle glow effects.  
  - **World 3 - Color Pulse:** Dynamic color pulses that change on ball hits.

- **Smooth Gameplay:** Responsive paddle controlled with arrow keys or A/D keys.  
- **Score Tracking:** Current and best scores displayed live.  
- **Achievements:** Unlock milestones at scores 5, 10, and 20 with visual pop-ups.  
- **Pause Functionality:** Press `P` to pause and resume the game.  
- **Countdown Timer:** 3-second countdown before game start.  
- **Game Over Screen:** Displays final score with retry and exit options.

---
